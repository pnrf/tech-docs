# Составление документов для участия в госзакупках

<Steps>
### Описание ситуации

Госзакупки имеют ряд особенностей. Например, в тех случаях, когда на рынок выводится инновационный продукт. 

Перед тем, как объявить конкурс, госзаказчику требуется разобраться в предметной области, оценить собственные потребности и сформулировать ожидания к закупаемому программному продукту. Для этого ведомства проводят открытые встречи с разработчиками, экспертами и общественными организациями.

### Задача

Для участия в таких встречах мне поручили составить документы, описывающие назначение, технические характеристики и возможности наших программных продуктов:

– ИИ-сервисов для выявления патологий на медицинских изображениях;  
– программного комплекса для автоматизации процессов разработки ИИ-сервисов.

### Результат

Задача выполнена. Требуемые документы составлены:

- [краткое описание программного комплекса ↗](https://drive.google.com/file/d/1QUUQ2VPHa6-k32d0_-uDtktQr2EoSj7D/view?usp=share_link)
- техническое задание (ТЗ) на комплексное программное обеспечение;
- три частных технические задания (ЧТЗ) на основные компоненты комплекса.

</Steps>
