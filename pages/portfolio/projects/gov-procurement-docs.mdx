# Документация для участия в госзакупках

<Steps>
### Описание ситуации

Госзакупки имеют ряд особенностей. Особенно в тех случаях, когда на рынок выводится инновационный продукт. Например, сервис искусственного интеллекта для выявления признаков патологий в легких. 

Перед тем, как объявить конкурс, государственному ведомству требуется разобраться в предметной области. Для этого проводятся открытые встречи с разработчиками, экспертами и общественными организациями.

### Задача

Для участия в такой встрече мне поручили составить документы, описывающие назначение и технические характеристики нашего продукта. За основу порекомендовали взять форму технического задания.

Задача выполнена. Составлены:
