# Примеры работ

<Callout type="warning" emoji="⚠️">
  Документы сейчас прорабатываю, чтобы не попасть под NDA. Могу предоставить при собесебовании.
</Callout>

## Технические документы

- Описание функциональных характеристик ПО
- Руковоство по установке и эксплуатации ПО
- Описание процессов жизненного цикла ПО
- Техническое задание (ТЗ) на комплексное программное обеспечение
- Частное технические задание (ЧТЗ) на компонент комплексного ПО
- Программа проведения клинических испытаний

## Юридические документы

- Лицензионный договор
- Договор возмездного оказания услуг с фрилансерами
- Пользовательское соглашение (на примере своего сайта)
- Политика конфиденциальности (на примере своего сайта)
- Информированное согласие владельца персональных данных
