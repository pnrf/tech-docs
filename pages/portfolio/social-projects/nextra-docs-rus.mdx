# Перевод документации Nextra

## Идея

В процессе создания своего [сайта-портфолио](https://pankratov.vercel.app) я выбрал фреймворк [Nextra](https://nextra.site). Он позволяет довольно быстро развернуть в сети базу с документацией. Причем, для этого не требуется углубленное знание языков программирования.

Однако, [официальная документация](https://nextra.site/docs) по Nextra доступна только на английском языке, что может стать камнем преткновения для русскоязычных пользователей. Я решил устранить это препятствие.

## Задача

Перевести официальную документацию Nextra на русский язык.

## Реализация

Одновременно с созданием собственного [сайта-портфолио](https://pankratov.vercel.app/portfolio/projects/my-portfolio-website) я приступил к реализации поставленной задачи.

## Результат

- Перевод доступен на моем сайте [по ссылке](https://pankratov.vercel.app/knowledge-base/nextra).

В будущем я планирую связаться с разработчиками Nextra, чтобы договориться с ними об официальной локализации их документации на русский язык. Но для этого от меня требуется более детально погрузиться в программный код этого продукта. Нужно время.

## Общественная польза

Надеюсь, что мой перевод поможет техническим писателям, которые хотят создать собственный сайт с документацией, но не владеют английским языком в должной мере.
