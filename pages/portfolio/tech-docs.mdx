# Техническая документация

У меня относительно небольшой опыт разработки технической документации. По большому счету он сводится к составлению документов для регистрации ИИ-сервисов в различных реестрах: реестре отечественного ПО **Минцифры России**, реестре медицинских изделий **Росздравнадзора** и реестре интеллектуальной собственности **Роспатента**.

ИТ-компания, в которой я работал, занималась разработкой сервисов искусственного интеллекта для медицинских целей в области лучевой диагностики, для выявления признаков паталогий на рентгеновских снимках.
