# Next.js ссылки

Все ссылки, содержащиеся в Markdown, автоматически преобразовываются в `Next.js links`. Это означает, что целевая страница будет заранее подготовлена. Когда вы кликнете по ссылке, эта страница будет отрисована на стороне клиента как SPA, т.е. без полной перезагрузки всей страницы. Например:

```md
Click [here](/about) to read more.
```

эквивалентно:

```jsx
import Link from 'next/link'

Click <Link href="/about">here</Link> to read more.
```
Подобный подход позволяет Nextra сделать навигацию между страницами быстрой и бесшовной.
