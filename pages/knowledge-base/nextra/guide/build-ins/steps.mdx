# Шаги

*Steps Component – встроенный компонент для преобразования нумерованного списка в визуальное представление шагов.*

## Пример

<Steps>

### Первый шаг

Текст с описанием первого шага.

### Второй шаг

Текст с описанием второго шага.

### Третий шаг

Текст с описанием третьего шага.

</Steps>

## Реализация

Внутри компонента `<Steps> </Steps>` обозначьте названия шагов заголовками третьего уровня (в markdown h3 – это `###`). В этом случае система их визуально преобразует.

<Steps>

### Первый шаг

Текст с описанием первого шага.

### Второй шаг

Текст с описанием второго шага.

</Steps>
