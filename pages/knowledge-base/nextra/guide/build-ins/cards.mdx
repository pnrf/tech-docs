## Реализация

import { Callout } from 'nextra/components'
 
<Callout type="warning" emoji="⚠️">
  В ряде случаев реализация, предложенная в [документации Nextra](https://nextra.site/docs/guide/built-ins/cards), не работает. Возникает сообщение об ошибке. В этом случае возможно вставить svg код иконки непосредственно в компонент `Card`, а не передавать ссылку на импортируемый svg файл.
</Callout>

### Согласно документации

{/* prettier-ignore */}
```mdx filename="Markdown"
import { Cards, Card } from 'nextra/components'
import { CardsIcon, OneIcon, WarningIcon } from '../../icons'

<Cards>

</Cards>
```
