# Docs-as-Code

*– концепция, предлагающая составлять документацию аналогично разработке программного кода.*

## Принципы

- использование системы контроля версий (например, git);
- проверка на ошибки с применением линтеров;
- переиспользование отдельных частей текста;
- использование [генераторов статичных сайтов ↗](https://jamstack.org/generators/).

## Легковесные языки разметки

- Markdown
- reStructuredText
- AsciiDoc
- LaTeX
- MakeDoc
