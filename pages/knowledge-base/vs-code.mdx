# VS Code

> **Visual Studio Code (VS Code)** — текстовый редактор, разработанный Microsoft для Windows, Linux и macOS. Позиционируется как «лёгкий» редактор кода для кроссплатформенной разработки веб- и облачных приложений. Имеет широкие возможности для кастомизации: пользовательские темы, сочетания клавиш и файлы конфигурации. Распространяется бесплатно.  
>

## Settings.json

Как и какие настройки использовать, смотрите на Youtube (например, в [уроке от WebDesign Master](https://youtu.be/JSGPd1E16-o?si=naQnUfRdt7ldc7LE)) или лучше читайте в [официальной документации](https://code.visualstudio.com/docs/getstarted/settings#_settingsjson).

Для доступа к файлу `settings.json` используйте комбинацию клавиш **`Ctrl + Shift + P`** (на MacOS: **`Cmd + Shift + P`**). Появится строка поиска. Введите запрос `settings.json`. В выпадающем списке выберите "Параметры: Открыть пользовательские настройки (JSON)" ("Preferences: Open User Settings (JSON)").

Мои настройки:

```json copy filename="settings.json"
 
{
	"liveServer.settings.donotShowInfoMsg": true,
	"workbench.startupEditor": "none",
	"liveServer.settings.CustomBrowser": "chrome",
	"editor.minimap.enabled": false,
	"breadcrumbs.enabled": false,
	"workbench.sideBar.location": "right",
	"window.zoomLevel": 1,
	"workbench.colorTheme": "New Moon Blue",
	"editor.smoothScrolling": true,
	"editor.fontSize": 14,
	"editor.tabSize": 2,
	"editor.detectIndentation": false,
	"editor.folding": false,
	"terminal.integrated.fontSize": 14,
	"editor.insertSpaces": false,
	"editor.linkedEditing": true,
	"editor.renderWhitespace": "boundary",
	"editor.parameterHints.enabled": false,
	"workbench.editor.enablePreview": false
}

```
